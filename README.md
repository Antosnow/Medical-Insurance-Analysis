# Medical Insurance Cost Prediction

## Project Overview

This project analyzes medical insurance customer data and predicts insurance charges using Machine Learning.

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-Learn
* Jupyter Notebook
* Power BI

## Dataset Features

* Age
* Gender
* BMI
* Children
* Smoker Status
* Region
* Insurance Charges

## Project Workflow

1. Data Loading
2. Data Cleaning
3. Exploratory Data Analysis (EDA)
4. Correlation Analysis
5. Feature Engineering
6. Machine Learning Model Training
7. Model Evaluation
8. Power BI Dashboard Creation

## Key Insights

* Smokers have significantly higher insurance charges.
* Age positively impacts insurance costs.
* Higher BMI is associated with higher medical expenses.
* Region has relatively low impact on insurance charges.

## Machine Learning Model

* Random Forest Regressor

## Evaluation Metrics

* MAE (Mean Absolute Error)
* RMSE (Root Mean Squared Error)
* R² Score
